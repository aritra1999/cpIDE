## CPIDE
A light weight crossplatform code editor with advanced features built for competitve programmers.

### Supported Languages
- C++
- C
- Python 
- Java
  
### Features
- Live autocomplete 
- 39 Themes
- Find and replace.

### Future featres
- Add your own templates 
- Question preview from the code editor.

### Default Shortcuts

| Command  | Shrotcut (Ctrl + ) |
|---|---|
| Save | s |
| Run  | Enter |
| Open  | o |
| Close | x |
| Find | f |
| Replace | h |
